copy_files()

takes the contents of the files hash and copies them from temp files and stores them in /temp_uploads. On upload, filename should get a random seed prefixed to the filename, but i‘ve added an extra 4 digit random seed to each filename just for extra protection. returns a hash with paths pointing to files as values. this is a workaround because you cant store files in the session variable.
