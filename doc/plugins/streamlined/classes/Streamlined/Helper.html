image_tag_if_exists(image, options = {})

Given an image file, checks to see if the image exists in the filesystem. If it does, display the image. If not, suppress the generation of the image tag. Used to add model-specific icons to the UI. If the icon does not exist, ensures no broken image tag or alternate text is rendered to the page.
