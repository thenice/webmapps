Module Streamlined::Helpers::HeaderHelper
In: vendor/plugins/streamlined/lib/streamlined/helpers/header_helper.rb

By default, headers are based on the class name (e.g. ‘Edit Person’) If your model class implements a name method, you will get that instead (e.g. ‘Edit John Doe’)
