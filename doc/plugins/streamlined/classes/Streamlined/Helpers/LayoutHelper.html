streamlined_side_menus()

Returns an array of arrays which define the side menus for the corresponding view. Override all side menus in all Streamlined controllers by defining this method in ApplicationHelper, or override on a controller by controller basis in their specific helpers. Each array should contain the text of the link and the URL parameters for the link.
